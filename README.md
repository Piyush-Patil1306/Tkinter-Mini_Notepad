# 📝 Tkinter Mini Notepad

A lightweight, minimal text editor built entirely using Python and the Tkinter GUI framework. It features a clean, responsive layout and a smart dynamic interface.

---

## ✨ Features

* **Minimalistic UI:** Clean `Segoe UI` typography with a modern color palette.
* **Smart "Clear" Button:** The button dynamically detects if text is present. It starts hidden, appears instantly when you begin typing, and vanishes the second the notepad is cleared or emptied.
* **Auto-Focus:** Automatically restores focus back to the text entry window after clearing so you can keep typing seamlessly.
* **Responsive Layout:** Stretches dynamically to look good at multiple window scales.

---

## 🛠️ Built With

* **Python 3**
* **Tkinter** (Python's built-in GUI library)
