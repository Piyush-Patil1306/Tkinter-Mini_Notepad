# 📝 Tkinter Mini Notepad

A simple and beginner-friendly text editor built using **Python** and the **Tkinter** GUI library. This project was created to practice GUI development, widget handling, layout management, and event-driven programming.

---

## ✨ Features

- 📝 Clean and minimal user interface
- 🎨 Modern layout with Segoe UI fonts
- 🗑️ Clear button with confirmation dialog
- 🔤 Font style selection (Normal, Bold, Italic)
- 🎯 Automatically returns focus to the text area after clearing
- 📐 Responsive layout that adjusts with the window size

---

## 🛠️ Built With

- Python 3
- Tkinter (Python's built-in GUI library)

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

### Run the Application

Clone the repository:

```bash
git clone https://github.com/Piyush-Patil1306/Tkinter-Mini_Notepad.git
```

Navigate to the project folder:

```bash
cd Tkinter-Mini_Notepad
```

Run the application:

```bash
python Notepad.py
```
